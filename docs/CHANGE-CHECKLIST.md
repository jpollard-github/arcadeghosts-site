# CHANGE-CHECKLIST

Use this checklist for any UI or layout change:

- [ ] Checked mobile at `375px` / `390px` / `430px`
- [ ] Checked tablet and desktop
- [ ] No horizontal scrolling
- [ ] Header/logo/hero does not overlap content
- [ ] Fixed/sticky controls do not block content
- [ ] CTA groups wrap cleanly
- [ ] New section/page uses shared responsive layout primitives where appropriate
- [ ] Review packet includes mobile screenshots for changed visual pages
- [ ] `MOBILE-TODO.md` updated if follow-up mobile work remains
