# Lead Generation Tracker Template

Purpose:

Use this as the column guide for the Google Sheet lead tracker.

Do not build a custom CRM yet.

## Columns

- Date Added
- Company
- Contact Name
- Role
- Email
- LinkedIn
- Website
- Category
- Source
- Signal / Why They Might Need Help
- Funnel Stage
- CTA Used
- UTM Link Used
- Outreach Sent
- Inquiry Form Clicked
- Discovery Link Sent
- Discovery Paid
- Last Contacted
- Follow-Up Date
- Follow-Up Count
- Outcome
- Notes
- Next Action

## Suggested Outcome Values

- Not contacted
- Contacted
- Replied
- Discovery call
- Proposal
- Won
- Lost
- Follow up later

## Suggested Funnel Stages

- Prospect
- Contacted
- Interested
- Intake Sent
- Discovery Proposed
- Discovery Paid
- Proposal Sent
- Won
- Lost
- Follow Up Later

## Suggested Notes

- Keep entries short and searchable.
- Record the specific reason the company looked like a fit.
- Track whether the next step is outreach, follow-up, wait, or archive.
- Keep the sheet simple. If Jason avoids using it, it is too complicated.
